Instructions to Run Prototype:

1. Unzip the folder.
2. Navigate into the project folder (Halley).
3. Run:  python src/main.py
   - This will generate sample billing output.
4. For insurance prototype, run:  python src/Insurance_main.py
5. A tests/ folder is included. To run tests:
   - Navigate into project folder and run: python -m unittest discover tests
